# CIL — Central Intelligence Library

Structured knowledge layer for RC Emporium.

## Tables / Card Types

- Ventures
- IP / invention cards
- Skills
- Documents
- Backlog tasks
- Deployment records
- Grant/funding records
- Security findings
- Decision logs

## Import Rule
Only import verified or clearly labeled data:

- `REAL_VERIFIED`
- `USER_PROVIDED_UNVERIFIED`
- `ESTIMATE`
- `TARGET`
- `PLACEHOLDER_BLOCKED`

No fake revenue, fake customers, fake deployments, or unsupported performance claims.
