# Manus Exports

Index location for Manus-generated outputs and export references.

## Rule
Manus is a builder/executor, not the master source of truth.

Every Manus output should be captured as:

1. Task prompt
2. Task URL / ID if available
3. Output artifact reference
4. Verification result
5. Decision: keep / revise / discard
6. Archive destination

Do not rely on Manus as the only storage location.
