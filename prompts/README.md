# Prompts

Prompt library for RC_OS command execution.

## Categories
- Bootstrap prompts
- Manus builder prompts
- OpenAI agent prompts
- Grant/application prompts
- RCBID audit/report prompts
- CIL import prompts

## Rule
Prompts should be versioned, specific, and truth-locked. Avoid prompts that ask agents to fake completion, claim access they do not have, or expose secrets.
